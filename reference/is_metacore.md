# Is metacore object

Is metacore object

## Usage

``` r
is_metacore(x)
```

## Arguments

- x:

  object to check

## Value

`TRUE` if metacore, `FALSE` if not

## Examples

``` r
# Loads in a metacore obj called metacore
load(metacore_example("pilot_ADaM.rda"))
is_metacore(metacore)
#> [1] TRUE
```
