# Check all data frames include the correct types of columns

This function checks for vector types and accepted words

## Usage

``` r
check_columns(
  ds_spec,
  ds_vars,
  var_spec,
  value_spec,
  derivations,
  codelist,
  supp
)
```

## Arguments

- ds_spec:

  dataset specification

- ds_vars:

  dataset variables

- var_spec:

  variable specification

- value_spec:

  value specification

- derivations:

  derivation information

- codelist:

  codelist information

- supp:

  supp information
